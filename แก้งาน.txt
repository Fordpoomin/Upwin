++++สร้าง QR code ++++

ตอนสร้าง แยก คณะ / กิจกรรม
- ให้ QR code คณะ มี 1 อัน ต่อ คณะ เท่านั้น*
- กิจกรรม เป็น QR code ชื่อกิจกรรม รายละเอียด ช่วงเวลาที่จัด วันที่เริ่มต้นของกิจกรรม วันที่สิ้นสุดกิจกรรม
ถ้าสิ้นสุดแล้ว จะแสกนเข้าไม่ได้*

Scan
-ก่อนแสกนจะไม่มีให้กรอกเบอร์
-กรณีแสกนซ้ำ QR Code เดิม การแสกนครั้งต่อไปจะเข้าสู่หน้าแสกนเลย ไม่มีให้กดยินยอนแล้ว


ไม่ให้ นิสิต ก็อปคิวอาร์โค้ดให้กันได้ ขณะไม่ได้อยู่ที่ QR code นั้นๆ*
Expire QRcode กิจกรรม( เช็คว่า สิ้นสุดกิจกรรมหรือยัง ถ้าสิ้นสุดแล้วไม่สามารถเช็คอินได้ )*



( ดูว่าใครเข้า คณะนี้บ้าง วันที่นี้มีใครเข้าบ้าง )
หน้ารายงาน Admin
หน้าสถิติตอบโจทย์ได้ว่า Admin ดูได้ว่าแต่ละตึก มีคนเข้า จำนวนคนเข้าเท่าไร โดยแยกวันเวลา แยกช่วงเวลาได้ โดยมี 
ฟิลเตอร์เป็นช่วงวันเวลาเราจะได้รู้ว่า ช่วงเวลาที่กำหนด มีใครเข้ามาบ้าง ( พิมพ์ค้นหา สิ่งที่จะค้นแล้วระบบจะฟิลเตอร์ออกมาเอง)*

เช่น
เมื่อวาน ที่ตึก Ict หรือกิจกรรม มีใครเข้ามาบ้าง พิมพ์ ชื่อตึก
แล้วลิส คนที่อยู่ใน Ict เฉพาะเช็คอิน ว่ามีใครเข้าร่วมกิจกรรม / หรือคณะนั้นๆ 
มีคนติดโควิดมานะ แล้วจะลิสชื่อคนที่ตึกนั้นออกมา ค้นหาชื่อตึกได้เลย *
